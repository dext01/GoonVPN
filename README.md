# GoonVPN

Быстрый и надёжный VPN-клиент для Android на основе протокола VLESS с Reality — современной технологии, которую невозможно заблокировать стандартными средствами DPI.

## Скачать

**[GoonVPN-v1.5.apk](https://github.com/dext01/GoonVPN/releases/download/v1.5/GoonVPN-v1.5.apk)** — установи на Android (разрешить установку из неизвестных источников)

> Требования: Android 8.0+

---

## Что это и как работает

Приложение создаёт зашифрованный туннель между твоим телефоном и VPN-сервером. Весь интернет-трафик идёт через него — провайдер и системы блокировок видят только случайный HTTPS-трафик, который не отличается от обычного посещения сайтов.

Внутри используется связка:
- **Xray-core** — маршрутизирует трафик по протоколу VLESS/Reality
- **HEV-socks5-tunnel** — поднимает TUN-интерфейс и перехватывает весь трафик устройства
- **Android VpnService** — системный API для создания виртуального сетевого интерфейса

---

## Возможности

- Подключение по VLESS/Reality — работает там, где обычные VPN заблокированы
- Добавление серверов через VLESS-ссылку или подписку (subscription URL)
- Статистика трафика в реальном времени (входящий / исходящий)
- Пинг серверов — показывает задержку до каждого
- Выбор DNS (Google, Cloudflare, AdGuard) — защита от DNS-утечек
- Настройки туннеля: MUX, фрагментирование пакетов, Xray TUN
- Просмотр логов прямо в приложении
- Тёмная и светлая тема

---

## Как добавить сервер

1. Открой приложение → нажми **+** в правом верхнем углу
2. Вставь VLESS-ссылку вида:
   ```
   vless://uuid@host:port?type=tcp&security=reality&pbk=...&sni=...#название
   ```
3. Или вставь URL подписки — приложение само скачает и добавит все серверы
4. Нажми кнопку подключения

## Telegram-бот

Для покупки подписки используй бота — [@vpngoon_bot](https://t.me/vpngoon_bot).

Бот автоматически создаёт VLESS-ключ на сервере и выдаёт его вместе с QR-кодом.

**Что умеет бот:**

- **Покупка подписки** — выбери тариф, спишутся монеты, придёт ключ и QR-код
- **Пополнение баланса** — через Telegram Stars (⭐), три варианта:
  - 150 монет — 120 ⭐
  - 400 монет — 300 ⭐
  - 1200 монет — 880 ⭐
- **Тарифы:**
  - 30 дней — 150 монет
  - 90 дней — 400 монет
  - 365 дней — 1200 монет
- **Мои подписки** — просмотр активных подписок, повторное получение ключа
- **Продление** — продлить любую активную подписку на любой тариф
- **Реферальная программа** — пригласи друга, получи 20 монет; друг получает 10 монет при регистрации
- **Уведомления** — бот предупреждает за 3 дня до истечения подписки и сообщает когда она закончилась

**Команды:**

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/buy` | Купить подписку |
| `/topup` | Пополнить баланс |
| `/balance` | Текущий баланс |
| `/mysubs` | Мои подписки |
| `/getkey` | Получить ключ повторно |
| `/ref` | Реферальная ссылка |

**Для администраторов** (панель `/admin`):
- Список всех пользователей и их балансов
- Статистика (кол-во юзеров, активных подписок, заработанных Stars)
- Просмотр состояния inbound на сервере
- `/add_balance <user_id> <монеты>` — начислить монеты вручную
- `/broadcast <текст>` — рассылка всем пользователям

## Структура проекта

```
app/src/main/java/com/goonvpn/app/
├── MainActivity.kt              — навигация, точка входа
├── ui/
│   ├── screens/
│   │   ├── HomeScreen.kt        — главный экран с кнопкой подключения
│   │   ├── ProxyScreen.kt       — список серверов
│   │   ├── SettingsScreen.kt    — настройки
│   │   ├── LogViewerScreen.kt   — просмотр логов
│   │   ├── ProfilesListScreen.kt
│   │   └── ProfileScreen.kt
│   ├── components/
│   │   └── AddServerBottomSheet.kt
│   └── theme/
│       ├── Color.kt             — цветовая схема
│       ├── Theme.kt
│       └── Type.kt
├── viewmodel/
│   └── VpnViewModel.kt          — состояние VPN, серверы, таймер, статистика
├── vpn/
│   ├── ServiceNet.kt            — VpnService, запуск HEV + Xray
```
